# LVINS

## 安装

- 安装[ROS 2](https://docs.ros.org/en/humble/index.html)（Humble）

- 安装依赖

```bash
sudo apt-get install git ros-dev-tools libmetis-dev python3-pip
```

- 自动补全依赖并编译

```bash
mkdir -p lvins_ws/src && cd lvins_ws/src && \
git clone https://github.com/LauZanMo/LVINS && \
bash LVINS/build.sh
```
